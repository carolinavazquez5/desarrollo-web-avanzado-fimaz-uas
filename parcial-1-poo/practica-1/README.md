# Práctica 1- Creación de clases y encapsulamiento en php

## Objetivo de la práctica
El objetivo de esta práctica es aprender a crear una clase en php y usar encapsulamiento para manejar los datos de un usuario.

## Descripción de la clase 
Se creó una clase llamada Usuario que tiene los datos: nombre y correo. estos datos son privados y se usan métodos getters y setters para poder obternelos o cambiarlos.

## Instrucciones de ejecución
1.- Abrir el proyecto en un servidor local.
2.- Ejecutar el archivo index.php 
3.- En el navegador se mostrarán el nombre y el correo del usuario.
