# Práctica 4- INTEGRACIÓN POO + HERENCIA + VALIDACIONES + EXCEPCIONES (PHP 8+)

## Objetivo
Hacer un pequeño sistema en PHP usando programación orientada a objetos para crear diferentes tipos de usuarios.

## Ruta de ejecución
Abrir en el navegador la siguiente ruta:
http://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-4/index.php

## Evidencia esperada
Al ejecutar el programa debe aparecer una tabla con los usuarios (Admin, Alumno e Invitado).  
También debe mostrarse un mensaje de error cuando se intenta crear un usuario con un correo mal escrito.
